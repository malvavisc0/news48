# Skill: Gather evidence before deciding

## Scope
Always active — all agents must ground reasoning in observable state.

## Rules
1. Run `news48` commands to gather evidence before making decisions.
2. Never assume system state — always verify.
3. Record what you observed, not what you expect.
4. Prefer evidence that maps directly to persisted fields or documented derived statuses.
5. If evidence does not cleanly support a claim, say that the claim is unproven.
